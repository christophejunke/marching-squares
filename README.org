Little game written during two days for [[https://itch.io/jam/lisp-game-jam-2018/][Lisp Game Jam 2018]] (not submitted).

Tested with SBCL:

    : apt install sbcl libsdl2-dev

Install Quicklisp as explained here: https://www.quicklisp.org/beta/.

    : (ql:quickload :marching-squares)

Use left and right arrows to move. Escape key restarts current level.

Depending on your screen, you might have to adjust =mc::*size*=.
